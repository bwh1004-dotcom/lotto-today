# 오늘 나에게 행운이 있었을까? 🎰

오늘의 하루를 기록하고, 나만의 로또 번호를 뽑는 감성 로또앱

## 메뉴 구성
- 🌅 오늘의 하루 — 기분·날씨·걸음수·식사·행운으로 번호 뽑기
- 🎆 불꽃놀이 — 탭 위치와 높이가 번호를 결정
- 🌙 꿈해몽 — 어젯밤 꿈으로 번호 해몽
- ⭐ 오늘의 운세 — 띠·기분으로 행운 번호 추출
- 🎡 행운의 휠 — 6번 스핀으로 번호 추출
- 🎰 슬롯머신 — 5번 스핀으로 번호 추출
- 🎛️ 나만의 공식 — 비율·홀짝·구간 직접 설정
- 💛 선호 번호 — 내가 좋아하는 번호 저장

## 기술 스택
- React 18 + Vite
- Web Audio API (효과음)
- localStorage (번호 저장)
